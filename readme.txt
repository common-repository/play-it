Play background sounds for your Wordpress site.

== Description ==

Play It is a plugin that enables you to play background sounds for your Wordpress site.  


== Installation ==

**Plugin Directory** 

1. Seach the plugin directory for plugin.
2. Click 'Install Now'.
3. Activate the plugin.
4. 'Play It' should be added to the side admin menu with the default settings.
5. You can modify the settings as needed.

**Upload** 

1. Download the plugin Zip file.
2. Go to plugins in the Wordpress Admin area.
3. Select 'Upload Plugin'
4. Activate the plugin.
5. 'Play It' should be added to the side admin menu with the default settings.
6. You can modify the settings as needed.
